# Dotfiles
### Warning: Use at your own risk! This repo is for my own, personal usecase.

Hey y'all, this is my second attempt at creating a NixOS dotfiles.
Programs used:
- **Operating System**: NixOS
- **Window Manager**: Hyprland
- **Shell**: Noctalia
- **File Manager**: Nemo
- **GTK Theme**: adw-gtk3
- **Icon Theme**: Papirus
- **Cursor**: Bibata-Modern-Classic
- **Wallpaper**: [Here](https://github.com/rhaps0dians/wallpapers).

Also, I put my configuration.nix and flake.nix in my $HOME, if you want to do the same, you can symlink it with:
``` 
sudo ln -s ~/configuration.nix /etc/nixos/configuration.nix
```
